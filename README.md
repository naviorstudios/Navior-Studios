# Navior Studios

Welcome to the ultimate premium, futuristic, and highly interactive e-commerce website for **Navior Studios**. Built to envision a ₹100 crore premium tech brand, this project features cinematic animations, a futuristic UI, strong storytelling, high performance, and a high conversion rate.

## 🚀 Features
- **Next.js App Router**: Modern and fast server-rendered application.
- **Vibrant & Futuristic Animations**: Crafted for maximum impression and smooth scrolling.
- **Premium E-Commerce Capabilities**: Dynamic product showcases, collections, and immersive shopping experience.

## 📁 Project Structure
The repository is divided into two primary sections:
1. `navior-studios/` - The main Next.js frontend and web application.
2. `backend/` - The backend processing APIs and database connectivity.

## 💻 Tech Stack
- Frontend: Next.js (React), Vanilla CSS (Custom design system), Typescript.
- Backend Integration: APIs built directly in Next.js `app/api`.
- Payments: Payment Gateway Integrations.

## 🛠️ Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/naviorstudios/Navior-Studios.git
   ```

2. **Navigate into the frontend directory:**
   ```bash
   cd navior-studios
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the local development server:**
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.
